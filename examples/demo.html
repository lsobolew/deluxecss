pixel-css — live-controllable CSS pixel art

The image below is pure CSS: layered linear-gradients whose colors are --color-* custom properties. Drag the pickers to recolor it live — no image is regenerated. That's the edge over a base64 background.

Animations

Every waterfall example below is generated from one source animation: monkey_island_waterfal.gif (640×286). Only the multi-layer variants render at that original size; the single-layer ones are downscaled (a single element can't paint a full-resolution frame).
