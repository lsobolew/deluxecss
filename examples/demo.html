deluxecss — live-controllable CSS pixel art

The image below is pure CSS: layered linear-gradients whose colors are --color-* custom properties. Drag the pickers to recolor it live — no image is regenerated. That's the edge over a base64 background.

Examples

Big Whoop — static image
big_whoop.png (632×144) as CSS: single-layer vs multi-layer, plus CGA/EGA/VGA palettes, next to the original PNG.
Stan — sprite animation
4-frame sprite (44×56) built from separate PNGs — the ideal case for frames mode: tiny and smooth.
Waterfall — color cycling
640×286 palette animation the efficient way: layered base, overlay only where pixels change, static colors inlined. Original GIF alongside.
Artificial Heart — IFF color cycling
Amiga .iff (ILBM) decoded natively; its CCRT palette-cycle range is reproduced as the exact CSS animation — no GIF, no sampling.
Matrix — frame limits ~193 MB
A few seconds of film, high color fidelity, multi-layer split across files. Where frame-by-frame animation strains. Run its gen.mjs first.
